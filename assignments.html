Homeworks

HW1

Design a static visualization for the given dataset (sketch on paper or use a digital design tool, such as PowerPoint / Keynote).

HW2

Use Tableau to explore the given dataset. Generate at least 5 hypotheses, and write a report (5 pages) to summarize your findings. Each hypothesis should be on a separate page with a paragraph of explanatory text and the visualization you used to answer your question.

HW3

Draw a graph of the given dataset.

HW4

Take home quiz. Answer the questions about data visualization principles.

HW5 ,

Write a 2-3 page essay to critique the given visualization. Discuss ways that the visualization can be used to mislead and the perceptual and cognitive reasons why.

Programming Assignments

P1

Static Visualization. Replicate the static visualization using Vega-Lite.

P2

Interactivity. Given the static starter code, add the following interactivity using Vega-Lite: hover to reveal a tooltip, implement the filter functions in the data, and click to add data to the list.

P3

Brushing and Linking. Given the starter code, add brushing and linking between the two views. Hovering over a point in one plot should highlight the associated points in the other.

Semester Group Project

[] Choose one of the provided datasets, and design and implement an interactive visualization using Vega-Lite. The programming assignments are designed to build up the skills necessary for this project.

In-Class Presentation

[] Sign up for a short in-class presentation of a paper of your choice from IEEE VIS. Summarize in 10-12 minutes: the problem that the paper addresses, the methodology, and the results.

Quizzes

[] Five short quizzes (10 min) will be administered in class on random, unannounced days. Your lowest grade will automatically be dropped.
